# Reto Tiburón V7.0 — Demo completa sin caché

Esta versión se abre en una dirección nueva:

https://christiangh7788-afk.github.io/reto-tiburon/demo-completa.html?v=70

`demo-completa.html` nunca existió en las versiones anteriores, por lo que
Chrome no puede sustituirla por la V5 guardada.

## Cómo probar

1. Abre `demo-completa.html?v=70`.
2. Pulsa **Entrar en modo demostración**.
3. El menú completo se abre automáticamente.
4. Debes ver **V7.0 · DEMO COMPLETA**.
5. Pulsa:
   - **Jugar Match‑3 ahora**
   - **Probar Funk de ventas**
   - **Centro de pruebas**

El flujo normal conserva validación con fotografía, mini misión,
celebraciones, rangos, 30 misiones y bonos.

Esta versión no instala service worker para evitar problemas de caché.
